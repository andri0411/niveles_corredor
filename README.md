## Juego simple con Flame

He añadido una implementación mínima de un juego de corredor usando `flame`.

- Archivos principales:
	- `lib/main.dart` — arranca el `GameWidget` y muestra los controles en pantalla.
	- `lib/game/runner_game.dart` — lógica del juego: jugador, suelo, movimiento y salto.

Controles (pantalla táctil):
- Mantener presionado el botón izquierdo: mover a la izquierda.
- Mantener presionado el botón derecho: mover a la derecha.
- Pulsar el botón central: saltar.

Ejecución:
1. Asegúrate de tener Flutter instalado y un emulador/dispositivo conectado.
2. En la ruta del proyecto, ejecuta:

```powershell
flutter pub get
flutter run
```

Preguntas o mejoras: puedo añadir sprites, colisiones más complejas o controles por teclado si lo deseas.
